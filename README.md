# Bug-tracking-
Bug tracking system built in Python to practice QA concepts and structured programming.

📌 Project Purpose

This project was created to:

Practice structured programming in Python

Work with lists and dictionaries for data modeling

Implement input validation and control flow

Simulate a basic bug lifecycle

Apply QA thinking to system design

It represents the first version of a system that will evolve with additional features and improvements.


📦 Bug Structure

Each bug is currently stored as a dictionary containing:

description

priority

status

All bugs are stored in a list during runtime.
